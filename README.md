# SQL Sales Analysis Project

## Project Overview
This project analyzes sales data using SQL to find useful business insights such as top customers, total revenue, and sales performance.

## Dataset
The dataset used in this project is taken from Kaggle.

## Tools Used
- SQL
- SQL Server
- Kaggle Dataset

## SQL Analysis Performed
- Data exploration
- Top 10 customers by sales
- Total sales calculation
- Sales by country
- Customer purchase analysis
- Product revenue ranking
- Average order value
- Top 10 products by revenue

## Project Structure 
sql → SQL queries used for analysis  
images → screenshots of SQL results  

## Conclusion
SQL was used to analyze the dataset and extract meaningful insights about customer behavior and sales performance.
